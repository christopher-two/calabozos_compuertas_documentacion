Tecnologías utilizadas:

  • Kotlin Multiplatform (KMP): Para compartir código entre las plataformas Android e iOS, maximizando la reutilización de la lógica de negocio y reduciendo el tiempo de desarrollo.

  • Ktor: Como cliente HTTP para manejar las comunicaciones con APIs externas de manera eficiente y asíncrona.

  • Firebase: Para servicios en la nube, incluyendo autenticación, almacenamiento y bases de datos en tiempo real, lo que permite una integración robusta y escalable.

  • Koin: Para la inyección de dependencias, facilitando la gestión y el testing del código al promover un diseño modular y desacoplado.

  • Coil: Para la carga y gestión eficiente de imágenes, optimizando el rendimiento y la experiencia del usuario.
